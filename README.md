# FlowerStory
> An Android Project about Flower Recognition

## Introduce
Flower Story 即“观花物语”

一款专为爱花识花养花而造的App，预计提供如下功能：

* 爱花：图片美化
> 在本地即可对图像进行美化处理
* 识花：花卉识别
> 对花卉进行拍照，并联网上传数据即可得到识别结果以及相关百科知识
* 养花：疾病识别
> 精确识别花卉病虫害，并提供解决方案仅供参考
